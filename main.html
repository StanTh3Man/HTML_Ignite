HTML basics!

This is a simple webpage that was written in html and styled with css!
html stands for Hyper text markup language.





Tools and Implementation of HTML

An HTML document is simply a regular text file that has been written in a way that your browser can interpret it and display it's elements.
As a result, any basic text editor may be used to create HTML files, such as notepad (which is included with Microsoft Windows),
or nano (a simple command-line editor included with most linux distros.).
Try to stay away from more sophisticated editors such as Microsoft Word, or Pages, as these editors may add additional formatting to the text; Which is not useful in this application.
Typically HTML documents describe the structure of a webpage, while css documents handle the styling of the page, and Javascript documents handle the scripting of the page.
We will only be looking at HTML in this tutorial; However, you may find more info on css here, as well as more info on Javascript here.
Tags start with ' < ' followed by the name of the tag, then ' > ' and are closed with ' </ ' followed by the name of the tag then ' > '.
The content of the tag goes between the opening and closing tags. ex. ' <p>This is my Paragraph!</p> '.
It is important to close tags when you are done with them, otherwise any new tags you open will be an element of the previous tag.
Some special tags do not need to be closed as they serve special functions. An example of this is the <br> tag; Which we will learn more about later.
Tags do not need to be given individual names in HTML as the tag simply tells the browser to create an element of a specific type; Meaning that you can use the same tag over and over without issue!
Tags result in the creation of an element, and elements are considered the building blocks of HTML pages.
In html, comments are started with the tag '<!--' and closed with the tag '-->' (without quotes'').
Comments are ignored by the web browser just lke in Python, C or Java, so they are purely for documentation and the readability of your code.

Getting started.

It is useful to set up a directory in an easily accessible place on your computer(such as the desktop), to store your HTML document, and all its supporting files in.
To get started with html you will need to create an HTML file. To do that, simply open your desired text editor, and select ' File -> Save-as ', then navigate to the
directory that you just created. You may name your file whatever you like, but be sure that your file ends in ' .html '.
Next we will need to start editing the file. Our first few lines of code will give the browser some information on what type of code it is reading and how to handle it, as well as the desired language, and encoding.
At the beginning of each html file, you must write the following code(sample code will be shown in red and example elements will be shown in green.):

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>

</body>
</html>

Adding elements to your web-page.

To add new elements, simply start adding tags to the ' <body> ' element that we created in the last step.
New elements should typically be added to ' <body> ' because elements in ' <head> ' are mostly meta-data(data that refers to other data).
Below we will go over some basic tags and how to add them to your web-page.

Headings

Headings may be added to your web-page by using the ' <h1> ' tag. Remember to close your tag with ' </h1> ' when you are done.
Heading tags range from ' <h1> ' to ' <h6> ' with the ' <h1> ' tag being the largest.

Below is an example of the HTML code required to make a heading element:

*Note the entire HTML document is shown here, but from now it will be assumed that you are adding elements to ' <body> '

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>

    <h1>Hello World!<h1>

</body>
</html>

If done properly Your heading element should look like this:

Hello World

Paragraphs

To start a paragraph use the <p> tag. Be sure to place it between the opening and closing body tags!
Once in a paragraph, you may use <br> to break to a new line. This is one of the special tags that we talked about earlier, so this tag does not need to be closed.
Once finished with your paragraph you must use the </p> tag to close it. When creating multiple pragraphs, a blank space is automatically inserted between them,
so there is no need to add a <br> to the end of your paragraphs.
Here is an example of a paragraph element:

<p>Put your text here,<br>and dont forget to break down to new lines!</p>

If done properly the paragraph element should end up looking like this:

Put your text here,
and dont forget to break down to new lines!

Links

You can also create links to other sites by using the tag '<a href="http://www.path.to.site.com">' and ending the tag with '</a>' (without quotes'')
The text that is located between the start and end of the tag is the text that will be highlighted. Links are generally created within paragaph elements.
For Example if you wanted to link to 'http://google.ca' your HTML code should look like this:

<p>
Click <a href="http://www.google.ca">HERE</a> to go to google
</p>

If done properly your element will look like this:

Click HERE to go to google

Images

You can also add images to your site with the ' <img src="path/to/image" width="desired w" height="desired h"> ' tag.
We need not worry about the placement of the image, because in HTML, elements are placed on the page and spaced automatically in the order that they are written.
The ' src="" ' parameter may either specify the path to the image on the hard disk, or it may specify the url of the image.
Keep in mind that the entire path to the image does not need to be specified, as long as the image is in the same directory as the HTML file,
or in a sub-directory of the HTML file's parent-directory.
Although the width and height options are not necessary for the tag, it is generally a good idea to specify them in order to avoid issues with
page formatting and over-sized images.

Here is an example of an image element, where the HTML file is located at ' /cmpt104/website.html ' and the desired image is located at ' /cmpt104/images/cats.png ':

<body>
<img src="images/cats.jpg" width="310" height="145">
</body>

*Notice that the entire path to the image was not specified! Only the relevant path after the HTML file's parent-directory is needed!
If this element was done correctly it should have resulted in an image that is 310 px wide and 145 px high as seen below.

Horizantal rulers

You may add a horizantal ruler to your site with the ' <hr> ' tag. This element is usefull for breaking up segments on your page. The ' <hr> ' tag does not need to be closed.
Below is some sample HTML code for creating a horizantal ruler:

<p>Verry important text<p>
<hr>
<p>Even more important text<p>

If done properly your element should look like this:

Verry important text

Even more important text

Changing font color

You can also change the text color quickly with the ' style="color:red" ' option within the ' <p> ' tag
This is not the most efficient or correct method of changing font color, as typically font color is handled by a separate css file; But for our purposes it will work.
You may either use basic colors such as 'yellow' or rgb colors by typing ' rgb(x,x,x) '
Below is some sample HTML code showing how to change font color.

<p style="color:yellow">Colored Text!<p>

If done properly your element should look like this:

Colored Text!

Putting it all togather and viewing your website!

Now that we have our elements, we can now go about viewing them in a browser. Navigate to the folder in which you have saved your HTML file, right click on it,
and select ' open with... ' then chose your web browser. The website should open, and you should see your elements! If some elements are missing, check your code for errors and ensure everything is spelled correctly and in the proper case.
HTML errors are not shown by your browser, so the browser will quietly ommit the element.
